# 地球防衛オペレーション 開発メモ

## アプリ概要
大輝さんの三日坊主をハックする「If-Then：地球防衛オペレーション」アプリ。
Gemini APIを使って毎日新しいキャラクターと壮大な並行世界ミッションを生成し、継続を促す。

## 確定した仕様

### キャラクターライフサイクル
- ミッション（目標＋If-Thenトリガー＋世界観）は一度だけ生成・固定保存
- **毎日アプリを開くたびに新しいキャラクターが登場**（熱血司令官・マッドサイエンティスト・迷子の子猫など）
- 同じミッションでも毎日違うキャラが届ける
- 「任務完了」「エネルギー不足（タイムリープ特異点）」どちらもそのキャラが応答

### データ設計
- Googleスプレッドシートで永続化
- **missionsシート**: id, goalText, ifThenTrigger, worldStory, createdDate, status
- **dailyLogsシート**: id, missionId, date, characterName, characterPersonality, result, responseText

### 3つのGeminiフェーズ
1. **ミッション作成時**: If-Thenトリガー＋世界観を生成（固定）
2. **毎日のキャラ生成**: 今日限りの新キャラ＋ミッション通達
3. **報告時**: 完了→世界救済絶賛 / サボり→タイムリープ特異点発動

### UIテーマ
- 背景: #0A0E1A（ネイビー）
- アクセント: #00D4FF（シアン）
- SF/サイバー系デザイン

## ファイル構成
- `Code.gs` — GASバックエンド（Gemini APIコール、Spreadsheet操作、doGet）
- `Index.html` — フロントエンドUI（SF/シアン系、タブ切り替え）

## GASデプロイ手順
1. GASプロジェクトで `Code.gs` と `Index.html` を貼り付け
2. Googleスプレッドシートを新規作成
3. スクリプトプロパティに設定：
   - `GEMINI_API_KEY` : Gemini APIキー
   - `SPREADSHEET_ID` : 作成したスプレッドシートのID
4. デプロイ → 新しいデプロイ → ウェブアプリ → アクセス: 自分のみ

## 科学的アプローチ（組み込まれた心理学）
- **If-Thenプランニング**: 「〇〇したら（If）→△△する（Then）」で行動トリガーを明確化
- **損失回避**: 「世界が崩壊する」という損失フレーミングで継続動機を強化
- **目標の最小化**: サボりそうな時は「1回だけ」という最小ハードルに下げる
